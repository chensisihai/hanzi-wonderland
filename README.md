# My React App

A clean React project built with Vite and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
  ├── App.tsx          # Main component
  ├── App.css          # App styles
  ├── index.css        # Global styles
  └── main.tsx         # Entry point
```

## Features

- React 19
- TypeScript
- Vite for fast development
- ESLint configured

Start building your app by editing `src/App.tsx`!
